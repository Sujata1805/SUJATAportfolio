Project Overview

News Application | Full Stack Web Development Project I developed a fully functional news application that allows users to browse, filter, and interact with the latest news articles. The application pulls real-time data using an external API to display news from various sources, with features that include: Responsive Design: Ensured seamless browsing across devices with a user-friendly interface built using JavaScript and Bootstrap. Dynamic Data Rendering: Integrated API calls to fetch and display the latest news articles, allowing users to filter by category or keyword. Backend: Utilized Django and PostgreSQL to manage user data, preferences, and article metadata. Authentication: Implemented user authentication for a personalized news experience. The project focused on delivering a clean, functional UI/UX while ensuring efficient backend performance, aiming for a smooth and engaging user experience.

Project Overview

This full-stack application manages medical records seamlessly.
                        Developed full-stack applications with a focus on both front-end and back
                        end technologies to deliver seamless, high-performing web solutions.
                        Utilized HTML, CSS, JavaScript, React and Bootstrap to design responsive,
                        user-friendly front-end interfaces, ensuring cross-browser compatibility
                        and enhanced user experience.
                        Built and maintained robust back-end systems using Python, Django, Java
                        and PostgreSQL, ensuring secure and efficient data handling for multiple
                        projects.
                        Integrated RESTful APIs for smooth communication between front-end and
                        back-end, optimizing data flow and enhancing application performance.
                        Designed and implemented role-based access control (RBAC) and multi
                        factor authentication (MFA) for secure user authentication and ensuring
                        data privacy and compliance with industry standards
